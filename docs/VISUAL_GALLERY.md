# Visual Gallery

This repository includes selected public-facing visuals that demonstrate the empirical workflow, climate-yield first stage, and wage pass-through mechanism.

## Included Figures

| File | What it demonstrates | Public-release role |
|---|---|---|
| `figures/selected/workflow_overview.svg` | End-to-end data flow from climate exposure construction to yield and wage regressions | Shows reproducible workflow organization |
| `figures/selected/temperature_response_by_rice_season.png` | Rice-season temperature-response relationship | Shows crop-calendar and agronomic exposure logic |
| `figures/selected/climate_yield_stage1_summary.png` | Selected climate-to-yield first-stage summary | Shows empirical first-stage diagnostics |
| `figures/selected/wage_yield_pass_through_by_contract.png` | Wage-yield relationship by meal-provision contract category | Shows the labor-market mechanism and heterogeneity focus |

## Why Only Selected Visuals Are Included

The full project contains additional manuscript figures, robustness checks, and appendix outputs. This public portfolio release includes only a curated visual set while excluding raw data, generated full tables, manuscript drafts, and complete thesis outputs.

## Reviewer Takeaway

The visual set is meant to signal three capabilities relevant to agricultural economics and climate-risk research:

- constructing agronomically meaningful climate exposure measures;
- connecting climate shocks to crop-yield outcomes;
- estimating and communicating pass-through from production risk to labor-market outcomes.
