# Example Extracts

These CSV files are small real-data extracts from processed outputs. They document expected columns and data shapes without releasing raw data, generated full panels, or complete result tables.

- `sample_climate_exposure.csv`: district-season climate exposure fields.
- `sample_rice_yield_panel.csv`: district-season rice yield panel.
- `sample_wage_panel.csv`: selected wage panel fields.
- `sample_analysis_panel.csv`: selected merged wage-yield analysis fields.
- `sample_contract_slope_diagnostic.csv`: compact wage-yield contract-slope diagnostic.

Values are intentionally limited to a few rows and should be treated as documentation examples, not as a complete replication dataset.
