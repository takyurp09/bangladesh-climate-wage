# Selected Figures

This folder contains a small set of public-facing visuals for the repository. These are included to make the GitHub portfolio easier to evaluate quickly.

Included figures:

- `workflow_overview.svg`: high-level reproducible workflow from climate data to wage regressions.
- `temperature_response_by_rice_season.png`: seasonal temperature-response visualization for rice yield.
- `climate_yield_stage1_summary.png`: selected Stage 1 climate-yield summary figure.
- `wage_yield_pass_through_by_contract.png`: selected wage-yield pass-through visualization by contract category.

Not included:

- full thesis result tables
- manuscript PDFs
- generated appendix tables
- raw data visualizations that reveal unpublished data products without context

The goal is to show the research workflow and coding output, not to release the full thesis chapter.
